Final Project – Information Security
This repository includes two main components:
Secure Password Generator
File Integrity Checker
Project Overview
The purpose of this project is to develop basic security tools that enhance system protection and data reliability.
The Secure Password Generator is designed to produce strong and unpredictable passwords based on user-defined criteria such as length and character types.
The File Integrity Checker is used to verify whether files have been modified by comparing their hash values using the SHA-256 algorithm.
Technologies Used
Linux Environment
Bash / Python
SHA-256 Hash Function
Repository Structure
password-generator/ → Contains the implementation of the password generator
file-integrity-checker/ → Contains the implementation of the integrity checker
