# Final Project – Information Security

This repository presents my work for the Information Security final project. It focuses on creating practical tools that strengthen system security.

## Project Components

- **Secure Password Generator**
- **File Integrity Checker**

## Project Description

The aim of this project is to design simple yet effective security tools that improve data protection and system reliability.

The **Secure Password Generator** allows users to generate strong and unpredictable passwords by selecting different options such as length and character combinations.

The **File Integrity Checker** helps detect any unauthorized modifications in files by generating and comparing hash values using the SHA-256 algorithm.

## Technologies

- Linux Environment  
- Bash / Python  
- SHA-256 Hashing  

## Repository Structure

- `password-generator/` → Implementation of the password generator  
- `file-integrity-checker/` → Implementation of the file integrity checker
